# Elite Robots CS SDK C

This repository provides a C language wrapper for the Elite Robots CS C++ SDK and uses **CMake** for build and installation.


## Requirements

- Upstream `elite-cs-series-sdk` C++ SDK
- CMake >= `3.16`
- C/C++ toolchain with C++17 support
