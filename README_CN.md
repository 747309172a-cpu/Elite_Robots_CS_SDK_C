# Elite Robots CS SDK C

本仓库为 Elite Robots CS C++ SDK 提供 C 语言封装接口，项目使用 **CMake** 进行构建与安装。


## 依赖要求

- 上游 `elite-cs-series-sdk` C++ SDK
- CMake >= `3.16`
- 支持 C++17 的 C/C++ 编译工具链
